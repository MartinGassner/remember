# Remember

## Prerequisites
You will need the following things properly installed on your computer.

* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/) (with NPM)
* [Bower](http://bower.io/)

## Installation

* `git clone git@github.com:MartinGassner/remember.git` 
* `cd remember
* `npm install`
* `bower install`
